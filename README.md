# MERN Blackjack Game

A full-stack Blackjack game built using React, Node.js, Express, and MongoDB Atlas.

--Features--
- Start new game
- Hit / Stand actions
- Score calculation
- Winner detection
- MongoDB Atlas integration

--Tech Stack--
Frontend: React  
Backend: Node.js, Express  
Database: MongoDB Atlas  

## Setup

---Backend
cd backend_
npm install
npm start

---Frontend
cd blackjack-client
npm install
npm start
